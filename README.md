# Claude Code Configurator

A visual configuration tool for [`.claude/settings.json`](https://code.claude.com/docs/en/settings) - the configuration file for [Claude Code](https://claude.ai/code).

**Live at: [configure-claude-code.vercel.app](https://configure-claude-code.vercel.app/)**

## Features

- Configure permissions with allow, deny, and ask rules
- Pre-built tool packs for common workflows (Git, Node.js, Docker, AWS, etc.)
- Web domain access packs for documentation sites
- Sandbox configuration
- MCP server settings
- Hooks configuration
- Import/export settings as JSON
- Share configurations via URL
- Undo/redo support (Cmd+Z / Cmd+Shift+Z)
